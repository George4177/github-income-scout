## Short

GitHub Income Scout finds and scores low-risk GitHub issue opportunities for practical open-source contribution.

## Medium

GitHub Income Scout is a small Python tool for developers who want a focused contribution plan. It screens public GitHub issues, rejects exploit work and fake engagement, and exports Markdown, CSV, and JSON reports for a Starter Audit.

## Profile Feature

### GitHub Income Scout

A practical issue research tool and Starter Audit workflow for developers who want to find low-risk GitHub contribution opportunities.

It helps screen for:

- Python automation
- GitHub Actions and CI fixes
- README and documentation work
- small developer tools
- data/report workflows

It avoids:

- spam PRs
- fake stars or reviews
- exploit abuse
- credential handling
- scraping behind logins
- platform bypasses
