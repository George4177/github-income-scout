# Service Reply Template

Use this when someone asks about the Starter Audit.

```text
Thanks for the interest.

The Starter Audit is a small fixed-scope GitHub opportunity report. I screen public GitHub issues against your skills and time budget, then return:

- 10 relevant issue opportunities
- top 3 recommended actions
- acceptance criteria and risk notes
- rejected risky matches when relevant
- one GitHub profile or README positioning suggestion

The goal is to help you choose practical work, not to guarantee income, merged PRs, sponsorship, or bounty awards.

To start, I need:

- your GitHub profile
- preferred languages/tools
- task types you want
- task types you want to avoid
- weekly time available
```
