## Social Preview / Pinned Repo Copy

```text
GitHub Income Scout is a small Python tool for finding low-risk GitHub issues worth pursuing. It filters out exploit work, fake engagement, vague tasks, and unsafe opportunities, then produces a Markdown report with difficulty, time, value, acceptance criteria, and failure risks.
```
