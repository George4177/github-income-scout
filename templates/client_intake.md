## Deliverables

Starter Audit deliverables:

- 10 screened GitHub issue opportunities
- top 3 recommended actions
- acceptance criteria and risk notes
- one GitHub profile or README positioning suggestion
- next-step checklist

## Acceptance

The audit is complete when the client receives a Markdown report with screened opportunities and a concrete next-step recommendation. It does not guarantee income, merged PRs, bounty awards, or sponsorship.
