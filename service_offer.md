# Service Offer Draft

## GitHub Opportunity Scout for Independent Developers

I help developers find realistic GitHub issues that match their skills and can support a stronger portfolio, better open-source visibility, or paid customization leads.

The service focuses on low-risk work:

- Python automation
- GitHub Actions and CI fixes
- documentation and README improvements
- small developer tools
- data analysis scripts
- maintainable workflow templates

I do not offer spam PRs, fake engagement, security exploitation, credential handling, scraping behind logins, or platform rule bypasses.

## Deliverables

- screened issue list with links
- difficulty and time estimates
- acceptance criteria summary
- failure risk notes
- safety-filter notes for rejected matches
- first-action recommendation
- optional README/profile positioning notes

## Best First Package

Starter Audit: USD 29-49

This package is designed to be small enough for a first sale and concrete enough to deliver in one working session.

## Client Requirements

The client should provide:

- GitHub profile link
- preferred languages or tools
- time available per week
- examples of projects they would like to work on

No passwords, private tokens, or payment account access are required.
