# Pricing Notes

These packages are positioned as practical developer services. They do not promise income, merged PRs, bounty awards, or sponsorship results.

## Starter Audit

Price idea: USD 29-49

Deliverables:

- 10 screened GitHub issues matched to the client's skills
- risk notes for each issue
- first-action checklist for the top 3 issues
- rejected high-risk/fake-engagement matches when relevant

Best for: developers who want a focused weekend contribution plan.

## Profile + Opportunity Pack

Price idea: USD 99-199

Deliverables:

- GitHub profile README review
- 1 repository README rewrite
- 15 screened issues
- contribution positioning notes
- lightweight service offer copy

Best for: developers who want their GitHub to support freelance leads.

## Weekly Scout

Price idea: USD 49-99 per month

Deliverables:

- weekly Markdown report
- configured query pack
- reviewed top 5 opportunities
- no outbound comments or PRs without approval
- safety-filter summary

Best for: small teams or solo developers who want a steady contribution funnel.

## Custom Automation

Price idea: quoted per project

Examples:

- private GitHub issue scout dashboard
- GitHub Actions cleanup
- README and documentation automation
- data report template generation

Terms should be simple: fixed scope, written acceptance criteria, no credential sharing, no platform abuse.
