# Service Menu

Small fixed-scope services for developers who want practical GitHub contribution leads, cleaner project presentation, or lightweight automation.

These services do not guarantee income, bounty awards, sponsorship, merged pull requests, stars, or client leads. The goal is to reduce wasted time and turn public GitHub research into a safer action plan.

## 1. GitHub Starter Audit

Best for: developers who want a short list of realistic GitHub issues matched to their skills.

Price idea: USD 29-49

Delivery time: usually one working session after intake is clear

Deliverables:

- 10 screened public GitHub issue opportunities
- top 3 recommended first actions
- task type, difficulty, and likely time cost
- acceptance criteria and failure-risk notes
- rejected-match notes for unsafe, noisy, claimed, or poor-fit work
- one GitHub profile or README positioning suggestion

Client provides:

- GitHub profile link
- preferred languages, tools, and task types
- weekly time available
- work boundaries and topics to avoid

Start here: [Starter Audit request](https://github.com/George4177/github-income-scout/issues/new?template=starter-audit.yml)

## 2. Profile + Opportunity Pack

Best for: developers who want their GitHub profile and one repository README to support freelance or open-source credibility.

Price idea: USD 99-199

Deliverables:

- GitHub profile README review
- one repository README rewrite plan or draft
- 15 screened issue opportunities
- contribution positioning notes
- direct request-link copy for a small service offer

Start here: [Profile + Opportunity Pack request](https://github.com/George4177/github-income-scout/issues/new?template=profile-opportunity-pack.yml)

Example delivery: [Profile + Opportunity Pack example](examples/profile_opportunity_pack_example.md)

Delivery checklist: [templates/profile_pack_delivery_checklist.md](templates/profile_pack_delivery_checklist.md)

Good fit:

- Python automation developers
- data/reporting script builders
- GitHub Actions and CI maintainers
- documentation and README contributors
- small developer-tool builders

## 3. Custom Automation

Best for: repeatable developer workflows that can be delivered as a small script, report generator, GitHub Action, or template.

Price: quoted after scope

Examples:

- GitHub issue scout dashboard or query pack
- recurring weekly scout reports
- README/report generator
- GitHub Actions cleanup
- Markdown/CSV/JSON/PDF report workflow
- data analysis template
- Codex-assisted workflow template

Start here: [Custom Automation request](https://github.com/George4177/github-income-scout/issues/new?template=custom-automation.yml)

## Boundaries

I do not work on:

- spam pull requests or mass maintainer comments
- fake stars, fake reviews, or fake engagement
- exploit abuse or unauthorized testing
- credential handling, private tokens, passwords, or payment account access
- scraping behind logins or platform-rule bypasses
- misleading income claims
