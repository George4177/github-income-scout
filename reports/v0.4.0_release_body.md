## Shareable HTML reports

This release adds a standalone HTML output for browser review, client delivery, and print-to-PDF workflows.

### Included

- CLI and Composite Action support for `format: html`
- responsive, print-friendly summary and opportunity details
- escaped dynamic content and HTTP(S)-only issue links
- HTML output in the weekly artifact workflow
- HTML files in the one-command Starter Audit delivery bundle
- generated examples for reviewing the format before installation
