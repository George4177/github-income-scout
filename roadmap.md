# Roadmap

## MVP

- Search public GitHub issues from configurable queries
- Score low-risk opportunities
- Reject exploit, fake-engagement, credential, and platform-abuse tasks
- Generate Markdown reports
- Support offline sample data for validation
- Include Starter Audit templates for service delivery

## Next

- Add repository health signals: recent commits, open PR volume, maintainer response time
- Add duplicate PR warning
- Add CSV export
- Add profile-specific scoring weights
- Add GitHub Actions workflow for scheduled reports

## Later

- Local HTML dashboard
- Historical tracking
- Saved decisions
- Client-ready PDF export
